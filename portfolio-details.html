Lazydist

Project information

  • Category: Automation Script
  • Language: Bash
  • Last Updated: 12 jan, 2023
  • Project URL: https://github.com/dedploit/lazydist

Automation Script to install tools from kali-linux to fresh debian based system and ubuntu. Here the script makes it easy to install tools without messing your distribution completely by upgrading from kali repo , mean while you can install the tools individualy even without typing the complete code everytime.
